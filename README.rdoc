= Redmine CKEditor plugin

This plugin adds the text formatting for using CKEditor to Redmine.

This fork from {a-ono / redmine_ckeditor}[https://github.com/a-ono/redmine_ckeditor] is adapted for Redmine v.4.2.3.

== What is the difference between a fork?
The deprecated {paperclip}[https://github.com/thoughtbot/paperclip] package has been replaced with an alternative supported {kt-paperclip}[https://rubygems.org/gems/kt-paperclip].

== What is CKEditor?

CKEditor is a WYSIWYG text editor.
See {the official site}[http://ckeditor.com/] for more details.
